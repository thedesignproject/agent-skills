## TEMPLATE STYLE NOTES

### Tone
- Conversational and confident, not corporate
- Use parenthetical asides and light humor to keep things human
- Write like a smart friend explaining something over coffee, not a whitepaper
- Match Dianne's preferred voice: first-person experience ("I've been testing..."), end with questions not CTAs on social

### Formatting Rules
- Headers use sentence case, not title case
- Bold only for key terms on first use or benefit headlines in lists
- Keep paragraphs to 3-4 sentences max
- Use bullet lists sparingly — only when listing 3+ parallel items
- Every section should be scannable in under 10 seconds

### SEO Checklist
- [ ] Primary keyword in title, H1, and first 100 words
- [ ] Slug is 3-5 keyword-rich words (no filler words like "the", "and", "for")
- [ ] 3-5 secondary/semantic keywords placed naturally in H2s and body
- [ ] Search intent identified and content depth matches top SERP results
- [ ] Featured snippet formatting: definition paragraphs (40-60 words), numbered lists for processes, tables for comparisons
- [ ] E-E-A-T signals: first-person experience, specific client examples, concrete metrics, 2-3 authoritative external links
- [ ] Alt text on all images
- [ ] Internal links to 3-5 related TDP pages
- [ ] External links to 2-3 authoritative sources
- [ ] Meta description written (under 160 characters)
- [ ] Table of contents with anchor links

### Content Quality Checklist
- [ ] Hook is specific and opinionated (not generic)
- [ ] Every example includes concrete metrics or outcomes
- [ ] "How" section is actionable enough that a reader could start today
- [ ] Post answers "so what?" for the target audience within the first 3 paragraphs
- [ ] Tone sounds like a person, not a brand
- [ ] No section exceeds 300 words without a visual break (image, list, or subheading)
