# Contributing

Thanks for your interest in contributing to Agent Skills! Here's how you can help.

## How to contribute

1. **Fork** the repo
2. **Create a branch** for your change (`git checkout -b my-contribution`)
3. **Make your changes** — add a new guide, fix a typo, improve an explanation
4. **Commit** with a clear message describing what you changed
5. **Open a pull request** against `main`

## What we're looking for

- Practical tips and patterns for AI-powered development
- Claude Code workflows, configurations, and integrations
- Corrections or improvements to existing content
- New guides on tools and techniques the community would find useful

## Guidelines

- Keep content clear and concise
- Include examples where possible
- One topic per file — keep things focused
- Test any code snippets or configurations before submitting

## Questions?

Open an issue if you have questions or want to discuss an idea before writing it up.
