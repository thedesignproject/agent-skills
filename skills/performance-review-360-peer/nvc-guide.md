# Non-Violent Communication (NVC) Guide for Feedback

Based on Marshall B. Rosenberg's framework. Use this as reference when helping someone refine their feedback.

## Core principle: Kind, not nice

NVC is not a tool for softening feedback. It's a tool for delivering hard truths in a way that can actually be heard. The goal is **candor with care**.

- **Nice** avoids discomfort to protect the giver. It's selfish.
- **Kind** tells the truth to serve the receiver's growth. It's generous.

Withholding feedback because it's uncomfortable denies someone the opportunity to improve. Every question in this process should push toward honesty, not comfort.

## The Four Components of NVC

### 1. Observations (not evaluations)
Describe what you saw or heard without mixing in judgment.

| Instead of (evaluation) | Try (observation) |
|---|---|
| "They're lazy" | "In the last month, they missed 3 sprint deadlines" |
| "They're a great communicator" | "During the incident last week, they sent clear updates every 30 minutes" |
| "They don't care about quality" | "The last two PRs they submitted had no tests" |

### 2. Feelings (not thoughts)
Express how the observation affected you emotionally. Feelings point inward, not outward.

| Instead of (thought disguised as feeling) | Try (actual feeling) |
|---|---|
| "I feel like they don't respect my time" | "I feel frustrated when meetings start late" |
| "I feel ignored" | "I feel discouraged when my suggestions aren't acknowledged" |

### 3. Needs (not strategies)
Identify the universal human need behind the feeling.

Common workplace needs: predictability, trust, autonomy, collaboration, clarity, recognition, growth, safety, efficiency, respect.

Example: "I need predictability around deadlines so I can plan my own work."

### 4. Requests (not demands)
Make a concrete, actionable, positive request (what you want, not what you don't want).

| Instead of | Try |
|---|---|
| "Stop being late to standups" | "Could you join standup by 9:05 so we can finish on time?" |
| "Don't push code without tests" | "Would you be willing to include at least one test per PR?" |

## Coaching tips for each feedback question

### "What do they do well?" — Help ground praise
- Ask for a specific moment: "When did you last notice this?"
- Ask for impact: "How did that affect the project or team?"
- Vague praise is kind but not useful. Specific praise is kind AND actionable.

### "What could they improve?" — The most important question
- If they lead with a label ("they're disorganized"), ask: "What specifically did you observe that gave you that impression?"
- If they describe impact on others, redirect to self: "How did that affect *you*?"
- If they can't think of anything, don't accept it. Everyone has growth areas. Push:
  - "If you could change one small thing about working with them, what would it be?"
  - "Think about a time something felt off, even slightly. What happened?"
  - "What would you want them to hear that nobody else is telling them?"
- If they're holding back: "Keeping this to yourself doesn't protect them — it means they never get the chance to work on it. What's the kind thing to say here?"
- NVC makes hard truths *hearable*. The goal is candor delivered with care, not avoidance dressed up as politeness.

### "Leadership principles?" — Help them connect behavior to values
- Don't ask about all 5. Ask "Which 1-2 principles come to mind when you think about this person?"
- Then ask for a specific story or example for each.

### "Work with them again?" — Honor the honest answer
- If they say "yes" easily, ask what makes them say that.
- If they hesitate, that hesitation is data. Help them name it: "What would need to be different for it to be a clear yes?"
- If the answer is "no" or "not sure," help them articulate why using observations and needs, not judgments. That's some of the most valuable feedback the person can receive.
